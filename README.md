# Spring Boot Form Validation

A Spring Boot project demonstrating server-side form validation using:

##Validation Annotation Used
- @NotBlank
- @Size
- @Email
- @AssertTrue
- @Valid
- BindingResult
- Thymeleaf

## Technologies
- Java
- Spring Boot
- Thymeleaf
- Maven
- Bean Validation (Jakarta Validation)

## Features
- User Name Validation
- Email Validation
- Terms & Conditions Validation
- Error Messages Display
